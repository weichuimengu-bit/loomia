# Loomia

> Weave your ideas into light.
> AIで、事業の断片を、ひとつの光に。

**福祉業界のためのAI伴走パートナー。**
訪問介護からスタートし、訪問看護・通所介護・ケアマネ・障害福祉まで、福祉業界全体のAI活用を伴走します。
AIが下書きし、専門職が確認・最終判断する設計。補助金活用は提携の社労士・行政書士と連携して支援します。

## Brand Voice

- **De-AI-fy**: AIっぽい誇大訴求(完全自動・全自動配信・自動判定等)を避け、Human-in-the-Loopの正確な表現を使う
- **Honest scope**: できないこと・対応しない領域を率直に開示する
- **専門職判断権の尊重**: AIはケアマネ・サ責・介護福祉士・看護職員を置き換えない

## Services (3階層モデル)

### Tier 1: 無料AI活用度診断
訪問介護事業所の現状を、AI活用の観点から診断。フォーム入力(所要1〜2分)→24時間以内にPDFレポート送付。

### Tier 2: 導入支援(初期構築)
診断レポートを元に、現場に最適なAI業務支援ツールを設計・導入。
- 初期構築: ¥100〜200万
- 補助金活用支援: 成功報酬 15〜20%(申請書類最終作成・官公署提出は提携士業)

### Tier 3: 月額継続伴走 + 5プロダクト群
- 訪問記録 音声入力AI
- 加算取得支援AI
- 訪問ルート最適化AI
- 家族連絡業務支援AI
- ケアプラン作成支援AI

月額: ¥5〜30万(規模・選択プロダクトにより変動)

## Coverage

### Loomiaが対応する領域
- 訪問介護事業所(現在主力)
- 訪問看護事業所(2026年Q4〜)
- 通所介護(2027年Q1〜)
- ケアマネ事業所(2027年Q2〜)
- 障害福祉サービス(2027年Q3〜)

### Loomiaが単独で受任しない領域(提携士業に送る)
- 補助金申請書類の最終作成・官公署への提出(行政書士・社労士独占)
- 厚労省管轄助成金申請(社労士独占)
- 医療判断・診断・投薬指示・SaMD該当機能(医師法・薬機法)
- ケアプランの最終決定・利用者への直接交付・同意取得(ケアマネ専門業務)
- 介護報酬の不正請求(介護保険法違反)
- 福祉以外の業界

## Compliance Absolutes

1. 補助金獲得=ゴール設計を避け、3か月伴走運用と現場研修をパッケージ化
2. 経営者向けデモではなく、現場ヘルパー定着を最優先
3. ハルシネーション対策: 出典明示・人間最終承認・限定スコープ運用
4. 「AIがケアマネを置き換える」訴求の完全排除
5. 要配慮個人情報は学習利用オフ設定のClaude API/Azure OpenAI Serviceでのみ取扱い
6. 士業独占業務は提携士業との分業
7. SaMD該当機能はPMDA一元相談を経ない限り実装しない
