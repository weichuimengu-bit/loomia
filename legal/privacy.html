Loomia(以下「当社」という。)は、Loomiaが提供する福祉事業者向けAI活用支援サービス(以下「本サービス」という。)を通じて取得する個人情報の保護に関し、個人情報の保護に関する法律(2026年1月施行の改正規定を含む。以下「個情法」という。)、関係省令、ガイドラインを遵守するとともに、本ポリシーに基づき個人情報の適切な取扱いに努めます。

第1条 取得する個人情報の項目

A. 直接お問い合わせいただく方からの取得

  • 氏名、所属事業所名、役職
  • メールアドレス、電話番号
  • お問い合わせ内容、ご要望

B. AI活用度診断システムを通じた取得

  • 事業所名、担当者名
  • メールアドレス
  • 業態、職員数、所在都道府県、利用者数
  • 使用中の介護記録ソフト、業務上の課題、加算取得状況、補助金経験、AI導入姿勢
  • 診断システムへの自由記述内容

C. 業務委託契約のお客様からの取得

業務委託契約に基づき委託される個人情報および要配慮個人情報の取扱いは、別途締結する「データ取扱合意書(DPA)」に従います。委託に関する取扱いの詳細は、本ポリシーではなくDPAをご参照ください。

D. ウェブサイト利用時の自動取得

  • IPアドレス、ブラウザ情報、リファラ、アクセス日時
  • Cookie、ローカルストレージに保存される識別情報(2026年改正法対応の同意取得を行います)

第2条 取得方法

当社は、利用者本人または当社の業務委託先(契約上、適切な秘密保持義務および安全管理措置を負う者)からのみ、個人情報を取得します。

当社は、要配慮個人情報を取得する場合、本人の事前の同意を取得するか、または個情法に定める例外事由に該当する場合に限ります。2026年改正法による医療・介護分野での同意例外緩和の適用を受ける場合は、その都度、ガイドラインに従い適正に取り扱います。

第3条 利用目的

当社は、取得した個人情報を、以下の目的のために利用します。

  • 本サービスの提供、運営、保守、改善のため
  • お問い合わせへの回答、個別のご相談対応のため
  • AI活用度診断レポートの作成および送付のため
  • 業務委託契約上の業務遂行のため
  • 当社サービスに関する情報、業界の最新動向に関する情報のメール配信のため(本人からの停止希望があれば速やかに停止します)
  • 利用統計の作成および分析(個人を特定しない形での集計に限ります)
  • 法令上の義務の履行、紛争への対応、不正利用の防止のため

当社は、本ポリシーに定める利用目的の範囲を超えて個人情報を利用しません。利用目的を変更する場合、変更後の利用目的を本人に通知または当社のウェブサイトで公表します。

第4条 要配慮個人情報の取扱い

絶対線: 当社は、要配慮個人情報を、業務委託契約上の業務遂行のためにのみ取り扱います。当社は、要配慮個人情報を、学習利用がオフに設定されていない汎用の生成AIサービス(無料版を含む)に入力しません。

当社が業務上利用する生成AIサービスは、Anthropic社のClaude API、Microsoft社のAzure OpenAI Serviceその他、入力データを学習に利用しない設定が可能であり、適切な商用契約またはエンタープライズ契約が締結されているサービスに限ります。

要配慮個人情報を生成AIに投入する前には、可能な限り匿名化または仮名化の前処理を行います。前処理が技術的に困難な場合は、利用者(委託元事業者)と協議のうえ取扱いを判断します。

第5条 第三者提供および委託

当社は、次の各号のいずれかに該当する場合を除き、利用者本人の同意を得ずに個人情報を第三者に提供しません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合
  • 委託先への提供(下記)

当社は、利用目的の達成に必要な範囲内において、個人情報の取扱いの全部または一部を業務委託先に委託することがあります。委託先には、安全管理措置を遵守する適切な事業者のみを選定し、契約により秘密保持義務を負わせます。代表的な委託先は以下の通りです。

  • 生成AI基盤モデル提供者: Anthropic社(Claude API)、Microsoft社(Azure OpenAI Service)
  • クラウドサービス提供者: Microsoft Azure、Google Cloud等
  • 決済代行業者(該当する場合)
  • 業務改善の支援を行う他の士業: 社会保険労務士、行政書士、税理士等(必要に応じて利用者の事前同意を得ます)

第6条 越境移転

当社が利用する生成AI基盤モデルおよびクラウドサービスのサーバが外国に所在する場合、利用者の個人情報が外国にある第三者に提供されることになります。

該当するサービスおよび所在国:

  • Anthropic社のClaude API: 米国(Amazon Web Services上で稼働)
  • Microsoft社のAzure OpenAI Service: 日本リージョン(東日本/西日本)を優先選択。やむを得ず外国リージョンを利用する場合は別途利用者にお知らせします。

当該外国にある第三者は、個情法および日本のガイドラインを上回る、または同等の個人情報保護措置を講じています。具体的には、SOC 2 Type II、ISO 27001、ISO 27018等の第三者認証を取得しており、データ処理契約(DPA)に基づき安全管理措置の遵守を契約上義務付けています。

第7条 安全管理措置

当社は、個人情報の漏洩、滅失、毀損を防止するため、以下の組織的、人的、物理的、技術的安全管理措置を講じます。

  • 組織的安全管理措置: 個人情報取扱責任者の設置、規程に基づく運用、定期的な点検
  • 人的安全管理措置: 業務委託先・取引先との秘密保持契約、教育・訓練
  • 物理的安全管理措置: 端末への施錠管理、アクセス権限管理、退職時のアカウント無効化
  • 技術的安全管理措置: アクセス制御、暗号化(保存時・通信時)、多要素認証、ログ取得・監視、脆弱性管理

第8条 本人からの開示等の請求

利用者本人は、当社が保有する自身の個人情報について、開示、訂正、追加、削除、利用停止、消去、第三者提供の停止を請求することができます。請求は、本ポリシー末尾のお問い合わせ窓口宛に書面または電子メールで行ってください。

当社は、本人確認のうえ、合理的な期間内(原則として請求受領後30日以内)に対応します。法令上、開示できない事項がある場合は、その旨を理由とともに通知します。

第9条 Cookieおよび類似技術

当社のウェブサイトは、利用者の利便性向上、利用統計の取得、サイト改善のため、Cookieおよび類似技術を使用することがあります。利用者は、ブラウザの設定でCookieの受入れを拒否することができますが、その場合、本サービスの一部の機能が利用できなくなることがあります。

2026年APPI改正に基づき、Cookie等を含む個人関連情報の取扱いについて、必要に応じて利用者の同意を取得します。本ウェブサイトで使用する主なCookieおよび類似技術は以下の通りです。

  • サイト利用状況の解析(Cloudflare Web Analytics等)
  • セキュリティ対策(不正アクセス検知)
  • ユーザー体験の改善

第10条 漏洩等発生時の対応

当社は、個人情報の漏洩、滅失、毀損その他事故が発生したときは、速やかに事実関係を調査し、二次被害の防止、再発防止のために必要な措置を講じます。

要配慮個人情報の漏洩等が発生した場合、または1,000人を超える本人の個人情報の漏洩等が発生した場合、その他個情法施行規則に該当する場合、当社は個人情報保護委員会への報告および本人への通知を、規則所定の期間内に行います。

第11条 本ポリシーの変更

当社は、関係法令の改正、本サービスの変更、その他必要に応じて本ポリシーを変更することができます。変更後の本ポリシーは、当社のウェブサイトに掲載した時点で効力を生じます。重要な変更については、利用者の同意を取得するか、合理的な手段により利用者に通知します。

第12条 お問い合わせ窓口

個人情報の取扱いに関するお問い合わせ、開示等の請求、苦情、ご意見等は、以下の窓口までお願いいたします。
